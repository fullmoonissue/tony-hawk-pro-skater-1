# TAS of All Known Gaps% of THPS 1

## Why All Known Gaps% and not All Gaps% ?

I've made a little investigation [here](./all-known-gaps.md) to conclude I'm not sure that all THPS 1 gaps have been found.

So, instead of calling the TAS category All Gaps% (as we can be sure about that for THPS 2 for instance), I prefer to be more reserved on the category name.
